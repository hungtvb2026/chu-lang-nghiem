# Chú Lăng Nghiêm – Học Thuộc

Ứng dụng học thuộc Chú Lăng Nghiêm bằng phương pháp **điền từ còn thiếu**.

## Tính năng

- 📖 **5 Đệ** — học từng đệ hoặc tất cả
- 🎯 **4 độ khó** — Dễ (15%) · Vừa (30%) · Khó (50%) · Siêu khó (80%)
- 💾 **Lưu cài đặt** vào localStorage (chọn đệ + độ khó)
- 🔀 **Làm mới** để xáo trộn từ bị ẩn
- ✅ **Gợi ý** hiện chữ cái đầu của từ
- 🎉 Banner hoàn thành khi điền đúng tất cả

## Chạy Local

```bash
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000)

## Deploy Vercel

```bash
npx vercel --prod
```

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Vercel**
